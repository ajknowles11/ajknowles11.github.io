About Me

Background

I was born in Huntington, New York, and I've lived there all my life until coming to Pittsburgh for school at Carnegie Mellon University. I grew up playing video games, and was introduced to programming when command blocks were added to Minecraft. Despite the fairly simple and limited scope, I fell in love with writing code that could have a noticeable effect on my gameplay.

In 2015, I became a bit more comfortable writing code when I made my first mod, for the game Cave Story. I don't have access to the actual files anymore, but I did record a small video. It was made as a joke between me and my brother, but it does contain custom tile and collision edits, and dialogue scripting.

I continued programming for my middle and high school robotics teams, using block coding for lego robots with the former and Java for the latter. My school did not actually teach computer science, so I mainly picked up programming skills from my older team members and online resources. My main focus at the time was math, since it was a subject that I did have classes for, and I was pretty good at it. Once I came to CMU, though, I found a new appreciation for computer science as my classes have been great. A year ago I also joined the Game Creation Society, and felt right at home making games, so that's what I've been doing a lot of lately.
